¡Bienvenido a ERP GTU Desarrollado por Sergio Rojas.

ERP GTU es un proyecto diseñado por mi para practicar y demostrar todas las habilidades para desarrollo full-stack que integra un backend .NET 8 Web API con Entity Framework Core, SQLite y un frontend Angular, implementando autenticación JWT (Bearer) y un modelo de datos complejo y completo para gestionar:

  • Usuarios y roles
  • Tareas con estados
  • Clientes
  • Productos
  • Pedidos y detalles de pedido
